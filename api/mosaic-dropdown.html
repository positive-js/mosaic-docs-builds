API reference for Angular Mosaic dropdown

import {McDropdownModule} from '@ptsecurity/mosaic/dropdown';

Additional classes

McDropdown

Properties
NameDescription

backdropClass: string

Class to be added to the backdrop element.

closed: EventEmitter<void | 'click' | 'keydown' | 'tab'>

Event emitted when the dropdown is closed.

direction: Direction

Layout direction of the dropdown.

hasBackdrop: boolean | undefined

Whether the dropdown has a backdrop.

items: QueryList<McDropdownItem>

List of the items inside of a dropdown.

overlapTrigger: boolean

Whether the dropdown should overlap its trigger.

panelClass: string

This method takes classes set on the host mc-dropdown element and applies them on the dropdown template that displays in the overlay container. Otherwise, it's difficult to style the containing dropdown from outside the component.

xPosition: DropdownPositionX

Position of the dropdown in the X axis.

yPosition: DropdownPositionY

Position of the dropdown in the Y axis.

Methods
focusFirstItem

Focus the first item in the dropdown.

Parameters

origin

FocusOrigin = 'program'

Action from which the focus originated. Used to set the correct styling.

resetActiveItem

Resets the active item in the dropdown. This is used when the dropdown is opened, allowing the user to start from the first option when pressing the down arrow.

McDropdownItem

This directive is intended to be used inside an mc-dropdown tag. It exists mostly to set the role attribute.

Properties
NameDescription

disabled: boolean

Methods
focus

Focuses the dropdown item.

Parameters

origin

FocusOrigin = 'program'

getLabel

Gets the label to be used when determining whether the option should be focused.

Returns
string

McDropdownContent

Dropdown content that will be rendered lazily once the dropdown is opened.

McDropdownTrigger

This directive is intended to be used in conjunction with an mc-dropdown tag. It is responsible for toggling the display of the provided dropdown instance.

Properties
NameDescription

data: any

Data to be passed along to any lazily-rendered content.

dir: Direction

The text direction of the containing app.

dropdown: McDropdownPanel

References the dropdown instance that the trigger is associated with.

dropdownClosed: EventEmitter<void>

Event emitted when the associated dropdown is closed.

dropdownOpened: EventEmitter<void>

Event emitted when the associated dropdown is opened.

opened: boolean

Whether the dropdown is open.

Methods
close

Closes the dropdown.

focus

Focuses the dropdown trigger.

Parameters

origin

FocusOrigin = 'program'

Source of the dropdown trigger's focus.

open

Opens the dropdown.

toggle

Toggles the dropdown between the open and closed states.

Additional interfaces
