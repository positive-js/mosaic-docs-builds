API reference for Angular Mosaic autocomplete

import {McAutocompleteModule} from '@ptsecurity/true/autocomplete';

Directives

McAutocomplete

Selector: mc-autocomplete

Exported as: mcAutocomplete
Properties
NameDescription
@Input()

autoActiveFirstOption: boolean

Whether the first option should be highlighted when the autocomplete panel is opened. Can be configured globally through the MC_AUTOCOMPLETE_DEFAULT_OPTIONS token.

@Input('class')

classList: string

Takes classes set on the host mc-autocomplete element and applies them to the panel inside the overlay container to allow for easy styling.

@Input()

displayWith: ((value: any) => string) | null

Function that maps an option's control value to its display value in the trigger.

@Input()

panelWidth: string | number

Specify the width of the autocomplete panel. Can be any CSS sizing value, otherwise it will match the width of its host.

@Output()

closed: EventEmitter<void>

Event that is emitted when the autocomplete panel is closed.

@Output()

opened: EventEmitter<void>

Event that is emitted when the autocomplete panel is opened.

@Output()

optionSelected: EventEmitter<McAutocompleteSelectedEvent>

Event that is emitted whenever an option from the list is selected.

id: string

Unique ID to be used by autocomplete trigger's "aria-owns" property.

isOpen: boolean

keyManager: ActiveDescendantKeyManager<McOption>

Manages active item in option list based on key events.

optionGroups: QueryList<McOptgroup>

options: QueryList<McOption>

panel: ElementRef

showPanel: boolean

Whether the autocomplete panel should be visible, depending on option length.

template: TemplateRef<any>

McAutocompleteTrigger

Selector: input[mcAutocomplete] textarea[mcAutocomplete]

Exported as: mcAutocompleteTrigger
Properties
NameDescription
@Input('mcAutocomplete')

autocomplete: McAutocomplete

The autocomplete panel to be attached to this trigger.

@Input('mcAutocompleteDisabled')

autocompleteDisabled: boolean

Whether the autocomplete is disabled. When disabled, the element will act as a regular input and the user won't be able to open the panel.

@Input('mcAutocompleteConnectedTo')

connectedTo: McAutocompleteOrigin

Reference relative to which to position the autocomplete panel. Defaults to the autocomplete trigger element.

activeOption: McOption | null

The currently active option, coerced to MatOption type.

onChange: (value: any) => void

View -> model callback called when value changes

onTouched: () => void

View -> model callback called when autocomplete has been touched

optionSelections: Observable<McOptionSelectionChange>

panelClosingActions: Observable<McOptionSelectionChange | null>

A stream of actions that should close the autocomplete panel, including when an option is selected, on blur, and when TAB is pressed.

panelOpen: boolean
