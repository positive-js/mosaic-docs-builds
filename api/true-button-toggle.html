API reference for Angular Mosaic button-toggle

import {McButtonToggleModule} from '@ptsecurity/true/button-toggle';

Additional classes

McButtonToggleChange

Change event object emitted by MсButtonToggle.

Properties
NameDescription

source: McButtonToggle

The MсButtonToggle that emits the event.

value: any

The value assigned to the MсButtonToggle.

McButtonToggleGroup

Exclusive selection button toggle group that behaves like a radio-button group.

Properties
NameDescription

buttonToggles: QueryList<McButtonToggle>

Child button toggle buttons.

change: EventEmitter<McButtonToggleChange>

Event emitted when the group's value changes.

controlValueAccessorChangeFn: (value: any) => void

The method to be called in order to update ngModel. Now ngModel binding is not supported in multiple selection mode.

disabled: boolean

Whether multiple button toggle group is disabled.

multiple: boolean

Whether multiple button toggles can be selected.

onTouched: () => any

onTouch function registered via registerOnTouch (ControlValueAccessor).

selected: any

Selected button toggles in the group.

value: any

Value of the toggle group.

vertical: boolean

Whether the toggle group is vertical.

Methods
emitChangeEvent

Dispatch change event with current selection and group value.

isPrechecked

Determines whether a button toggle should be checked on init.

Parameters

toggle

McButtonToggle

isSelected

Checks whether a button toggle is selected.

Parameters

toggle

McButtonToggle

syncButtonToggle

Syncs a button toggle's selected state with the model value.

Parameters

toggle

McButtonToggle

Toggle to be synced.

select

boolean

Whether the toggle should be selected.

isUserInput

boolean = false

Whether the change was a result of a user interaction.

McButtonToggle

Single button inside of a toggle group.

Properties
NameDescription

buttonToggleGroup: McButtonToggleGroup

change: EventEmitter<McButtonToggleChange>

Event emitted when the group value changes.

checked: boolean

Whether the button is checked.

disabled: boolean

mcButton: McButton

type: ToggleType

value: any

McButtonToggleGroup reads this to assign its own value.

Methods
focus

Focuses the button.

markForCheck

Marks the button toggle as needing checking for change detection. This method is exposed because the parent button toggle group will directly update bound properties of the radio button.

onToggleClick

Checks the button toggle due to an interaction with the underlying native button.
